# organizador_de_termos
Visando a organização, praticidade e facilidade, estou desenvolvendo um sistema de RPA(Robotic Process Automation)  capaz de organizar arquivos .pdf de termos assinados de responsibalidades sobre equipamentos da empresa. na versão beta o sistema contará com algumas funções como backup, organizador por departamentos e renomeador padronizado com IA.
